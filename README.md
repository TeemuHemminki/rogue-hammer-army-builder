This is a fan made army builder for "Rogue Hammer. Gothic scifi miniatures gaming" (https://www.wargamevault.com/product_reviews.php?products_id=418767) by Ivan Sorensen / Nordic Weasel Games.

Project principles:
-Only pure html5 methods with no frameworks.
-No LLM generated code. But LLM can be used for solving problems or design patterns.
